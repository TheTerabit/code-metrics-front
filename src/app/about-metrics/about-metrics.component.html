About metrics

Methods This metric shows the number of defined methods in the class. The number of methods with concrete modifier is summed.
Fields This metric shows the number of defined fields in the class. The number of fields with concrete modifier is summed.
Cyclomatic Complexity This metric is the quantitative measure of the number of linearly independent paths in it. It is a software metric used to indicate the complexity of a program.
Number of code lines This metric shows number of lines in class and in methods defined inside the class.
